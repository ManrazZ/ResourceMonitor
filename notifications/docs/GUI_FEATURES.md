# Документация по графическому интерфейсу

## Обзор

Графический интерфейс ResourceMonitor предоставляет визуальное отображение метрик системы в реальном времени.

## Элементы интерфейса

### Заголовок
- Отображает название приложения.

### Панели метрик
- **CPU Usage**: показывает текущую загрузку процессора в процентах.
- **Memory Usage**: показывает текущее использование оперативной памяти в процентах.
- **Disk Usage**: показывает текущее использование дискового пространства в процентах.

Каждая панель содержит:
- Числовое значение в процентах
- Прогресс-бар с цветовой индикацией

### Цветовая индикация
- Зеленый: значение ниже 60%
- Желтый: значение между 60% и 80%
- Красный: значение выше 80%

### Кнопки управления
- **Start Monitoring**: запускает мониторинг.
- **Stop Monitoring**: останавливает мониторинг.

### Время последнего обновления
- Отображает время последнего обновления метрик.

## Использование

1. Запустите приложение.
2. Нажмите "Start Monitoring" для начала сбора данных.
3. Метрики будут обновляться каждые 2 секунды.
4. Для остановки нажмите "Stop Monitoring".

## Примечания

- Приложение использует многопоточность, поэтому интерфейс не блокируется во время сбора данных.
- Для закрытия приложения используйте крестик окна или остановите мониторинг и закройте окно.
