# ResourceMonitor

Система мониторинга системных ресурсов.

## Описание
ResourceMonitor предназначен для мониторинга CPU, памяти, диска и сети
в реальном времени с возможностью анализа и уведомлений.

## Функционал
- Сбор метрик CPU, памяти, диска, сети
- Анализ пороговых значений
- Уведомления при превышении лимитов

## Стек технологий
- Python 3.10+
- psutil
- logging
